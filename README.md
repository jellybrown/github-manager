# Github Manager

## 소개

- github repository를 간편하게 즐겨찾기 할 수 있습니다.<br>
- 즐겨찾기한 repository 페이지에서는 issue를 볼 수 있습니다.

<br>

## 사용 언어 및 라이브러리

- Typescript, React, Styled-components

<br>

## 사용법

- `git clone https://github.com/jellybrown/github-manager.git`

- `npm install`

- `npm run start`

<br>

## 구현 기능

- [x] repository 즐겨찾기 (최대4개)
- [x] issue state 필터
- [x] issue pagination
